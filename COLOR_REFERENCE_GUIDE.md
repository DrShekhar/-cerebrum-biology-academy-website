## Shadow System Hierarchy

### Shadow Sizes & Opacities

#### Level 1: Subtle (Base State)

```css
shadow-sm shadow-{color}-500/20
```

- Used for: Small elements, inactive states
- Opacity: 20%
- Size: Small shadow

#### Level 2: Standard (Default State)

```css
shadow-lg shadow-{color}-500/20
shadow-xl shadow-{color}-500/20
```

- Used for: Cards, containers, buttons
- Opacity: 20%
- Size: Large or extra-large

#### Level 3: Elevated (Hover State)

```css
hover:shadow-xl hover:shadow-{color}-500/30
hover:shadow-2xl hover:shadow-{color}-500/30
```

- Used for: Hover states
- Opacity: 30%
- Size: Extra-large or 2XL

#### Level 4: Prominent (Active/Focus State)

```css
hover:shadow-2xl hover:shadow-{color}-500/40
```

- Used for: Active states, important elements
- Opacity: 40%
- Size: 2XL

## Accessibility & Contrast

### WCAG AAA Compliant Pairings

#### Dark Text on Light Backgrounds

- All gradient backgrounds (50-shades) with gray-800/900 text
- Contrast ratio: > 7:1

#### White Text on Gradient Buttons

- All 500+ shade gradients with white text
- Contrast ratio: > 4.5:1

#### Shadow Visibility

- Minimum opacity: 20% for visibility
- Maximum opacity: 40% to avoid overwhelming

## Quick Copy-Paste Reference

### Complete Gradient + Shadow Patterns

#### Pattern 1: Button/Card

```tsx
className="
  bg-gradient-to-r from-blue-500 via-indigo-500 to-purple-600
  shadow-lg shadow-blue-500/30
  hover:shadow-xl hover:shadow-indigo-500/40
  transition-all duration-300
"
```

#### Pattern 2: Icon Container

```tsx
className="
  bg-gradient-to-r from-green-400 via-teal-500 to-cyan-600
  shadow-lg shadow-green-500/30
  hover:shadow-xl hover:shadow-cyan-500/40
  hover:ring-2 hover:ring-teal-400 hover:ring-offset-2
  transition-all duration-300
"
```

#### Pattern 3: Activity Icon

```tsx
className="
  bg-gradient-to-br from-purple-100 to-pink-100
  shadow-sm shadow-purple-500/20
  hover:shadow-md hover:shadow-purple-500/30
  transition-all duration-300
"
```

#### Pattern 4: Section Container

```tsx
className="
  bg-white/60
  backdrop-blur-sm
  shadow-xl shadow-indigo-500/10
  hover:shadow-2xl hover:shadow-indigo-500/20
  transition-all duration-300
"
```
