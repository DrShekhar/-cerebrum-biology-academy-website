# NEET 2026 Eligibility Criteria: Complete Guide

Before starting your NEET preparation, it's crucial to ensure you meet all eligibility requirements. This comprehensive guide covers every aspect of NEET 2026 eligibility - from age limits to educational qualifications.

---

## Quick Eligibility Checklist

| Criteria              | Requirement                   |
| --------------------- | ----------------------------- |
| Nationality           | Indian/NRI/OCI/PIO/Foreign    |
| Age (Minimum)         | 17 years by Dec 31, 2026      |
| Age (Maximum)         | No upper limit                |
| Qualification         | 10+2 with PCB                 |
| Minimum Marks in 10+2 | 50% (General), 40% (Reserved) |
| Attempts              | Unlimited                     |

## Detailed Eligibility Criteria

### 1. Nationality Requirements

#### Who Can Apply for NEET 2026?

| Nationality                     | Eligibility | Documents Required                      |
| ------------------------------- | ----------- | --------------------------------------- |
| Indian Citizens                 | Yes         | Aadhaar, Class 10 certificate           |
| NRI (Non-Resident Indian)       | Yes         | Valid Indian passport, NRI certificate  |
| OCI (Overseas Citizen of India) | Yes         | OCI card, passport                      |
| PIO (Person of Indian Origin)   | Yes         | PIO card, passport                      |
| Foreign Nationals               | Yes         | Valid passport, eligibility certificate |

**Important:** Foreign nationals and NRIs compete for 15% seats reserved under NRI/Supernumerary quota.

### 2. Age Criteria

#### Minimum Age Requirement

| Criterion           | Requirement       |
| ------------------- | ----------------- |
| Minimum Age         | 17 years          |
| Date of Calculation | December 31, 2026 |
| Born on or before   | December 31, 2009 |

**Example:** If you were born on January 1, 2010, you are NOT eligible for NEET 2026.

#### Maximum Age Limit

| Category       | Upper Age Limit |
| -------------- | --------------- |
| All Categories | No upper limit  |

**Note:** The Supreme Court removed the upper age limit. Previously, there was a 25-year limit for General and 30-year limit for Reserved categories.

### 3. Educational Qualification

#### Required Subjects in Class 12

| Subject               | Requirement |
| --------------------- | ----------- |
| Physics               | Mandatory   |
| Chemistry             | Mandatory   |
| Biology/Biotechnology | Mandatory   |
| English               | Mandatory   |
| Any other subject     | Optional    |

**Important Notes:**

- Biology is mandatory (Biotechnology alone is NOT sufficient for some colleges)
- Mathematics is NOT required
- Practical marks are included in percentage calculation

#### Minimum Marks in Class 12 (PCB)

| Category      | Minimum Marks | Condition        |
| ------------- | ------------- | ---------------- |
| General (UR)  | 50%           | In PCB aggregate |
| General-EWS   | 50%           | In PCB aggregate |
| OBC (NCL)     | 40%           | In PCB aggregate |
| SC            | 40%           | In PCB aggregate |
| ST            | 40%           | In PCB aggregate |
| UR-PwD        | 45%           | In PCB aggregate |
| OBC/SC/ST-PwD | 40%           | In PCB aggregate |

**PCB Aggregate Calculation:**

```
PCB Percentage = (Physics + Chemistry + Biology) / 3
```

For example: Physics: 80, Chemistry: 75, Biology: 85
PCB Percentage = (80 + 75 + 85) / 3 = 80%

#### Appearing Candidates

Students appearing for Class 12 exams in 2026 are eligible to apply. They must:

- Submit proof of passing before counselling
- Meet minimum marks requirement
- Provide original documents during verification

### 4. Number of Attempts

| Aspect            | NEET 2026 Rule       |
| ----------------- | -------------------- |
| Maximum Attempts  | Unlimited            |
| Gap Years Allowed | Yes, unlimited       |
| Re-attempt        | Yes, no restrictions |

**Note:** You can appear for NEET multiple times until you get admission or achieve desired score. There is no restriction on the number of attempts.

## Category-wise Eligibility Details

### General Category

| Requirement           | Details        |
| --------------------- | -------------- |
| Minimum marks in 12th | 50% in PCB     |
| Age                   | 17+ years      |
| Reservation           | No reservation |
| Income Limit          | None           |

### EWS (Economically Weaker Section)

| Requirement           | Details                     |
| --------------------- | --------------------------- |
| Minimum marks in 12th | 50% in PCB                  |
| Family Income         | < ₹8 lakhs per annum        |
| Land Holding          | < 5 acres agricultural land |
| House                 | < 1000 sq ft in town/city   |
| Reservation           | 10% seats                   |

**Documents Required:**

- EWS certificate from Tehsildar/SDM
- Income certificate
- Property documents (if any)

### OBC (Non-Creamy Layer)

| Requirement           | Details                          |
| --------------------- | -------------------------------- |
| Minimum marks in 12th | 40% in PCB                       |
| Creamy Layer          | Must NOT belong to creamy layer  |
| Income Limit          | < ₹8 lakhs (creamy layer cutoff) |
| Reservation           | 27% seats                        |

**Documents Required:**

- OBC-NCL certificate from competent authority
- Should be in central list of OBCs

### SC (Scheduled Caste)

| Requirement           | Details    |
| --------------------- | ---------- |
| Minimum marks in 12th | 40% in PCB |
| Income Limit          | None       |
| Reservation           | 15% seats  |

**Documents Required:**

- SC caste certificate from District Magistrate
- Should be in state/central SC list

### ST (Scheduled Tribe)

| Requirement           | Details    |
| --------------------- | ---------- |
| Minimum marks in 12th | 40% in PCB |
| Income Limit          | None       |
| Reservation           | 7.5% seats |

**Documents Required:**

- ST certificate from District Magistrate
- Should be in state/central ST list

### PwD (Persons with Disability)

| Requirement           | Details                   |
| --------------------- | ------------------------- |
| Minimum marks in 12th | 45% (UR), 40% (Reserved)  |
| Disability Percentage | Minimum 40%               |
| Reservation           | 5% horizontal reservation |

**Eligible Disabilities:**

- Blindness and low vision
- Deaf and hard of hearing
- Locomotor disability (including cerebral palsy)
- Leprosy cured
- Dwarfism
- Acid attack victims
- Muscular dystrophy
- Multiple sclerosis
- Mental illness
- Autism spectrum disorder
- Specific learning disabilities
- Multiple disabilities

**Documents Required:**

- Disability certificate from authorized medical board
- Benchmark disability certificate (40%+)

## State Quota Domicile Requirements

### What is Domicile?

Domicile is proof of residence in a particular state, required for 85% state quota seats.

### Common Domicile Requirements

| Criterion         | Requirement                           |
| ----------------- | ------------------------------------- |
| Education         | Studied 10th, 11th, 12th in the state |
| Residence         | Parents residing in state for X years |
| Birth             | Born in the state                     |
| Parent Employment | Parent employed in state government   |

### State-wise Domicile Rules

| State          | Primary Criterion               |
| -------------- | ------------------------------- |
| Maharashtra    | 15 years residence OR education |
| Karnataka      | 7 years residence OR 10th-12th  |
| Tamil Nadu     | Nativity OR 5 years residence   |
| Delhi          | 3 years education in Delhi      |
| Uttar Pradesh  | Domicile certificate            |
| Gujarat        | 10 years residence OR education |
| Madhya Pradesh | Domicile certificate            |
| Rajasthan      | Bonafide resident certificate   |

**Tip:** Check your state's specific domicile requirements well in advance.

## Frequently Asked Questions

### Can I appear for NEET without Biology in 12th?

No. Biology/Biotechnology is mandatory for NEET. If you have only taken Biotechnology without Biology, verify with specific colleges as some may not accept it.

### Is there any upper age limit for NEET 2026?

No. The Supreme Court has removed the upper age limit. Anyone who is 17+ years old can appear for NEET.

### Can I apply for NEET if I scored 49% in PCB (General)?

No. General category requires minimum 50% in PCB. You can either:

1. Appear for improvement exam
2. Check if your state board provides grace marks

### How many times can I attempt NEET?

There is no limit on the number of attempts. You can appear for NEET any number of times.

### Is Aadhaar mandatory for NEET?

Aadhaar is required for Indian citizens. NRIs and foreign nationals can use passport instead.

### Can NIOS students apply for NEET?

Yes. NIOS students with Physics, Chemistry, and Biology (with practicals) can apply for NEET.

### What if I don't have English in Class 12?

English is a mandatory subject for NEET. If you don't have English, you may need to appear for English as an additional subject.

### Is NEET eligibility same for government and private colleges?

Yes, the basic eligibility criteria are same. However, private colleges may have additional requirements for management/NRI quota seats.
