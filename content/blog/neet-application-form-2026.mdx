## Step-by-Step Application Process

### Step 1: Registration

1. Visit official website: **neet.nta.nic.in**
2. Click on "New Registration"
3. Read instructions carefully
4. Click "Proceed to Apply Online"

#### Information Required:

| Field               | What to Enter               |
| ------------------- | --------------------------- |
| Candidate's Name    | As per Class 10 certificate |
| Mother's Name       | As per official records     |
| Father's Name       | As per official records     |
| Date of Birth       | As per Class 10 certificate |
| Gender              | Male/Female/Transgender     |
| Mobile Number       | Active number (for OTPs)    |
| Email ID            | Active email                |
| Nationality         | Indian/NRI/OCI/Foreign      |
| Identification Type | Aadhaar/Passport            |
| ID Number           | 12-digit Aadhaar/Passport   |

**After Registration:**

- System generates **Application Number**
- Password sent to mobile and email
- **SAVE THESE CREDENTIALS**

### Step 2: Fill Application Form

#### Personal Details

| Field                | Instructions                           |
| -------------------- | -------------------------------------- |
| Category             | Select carefully - cannot change later |
| Sub-category         | If applicable (PwD, EWS)               |
| State of Eligibility | Where you passed 12th                  |
| Area                 | Rural/Urban                            |
| Religion             | Optional                               |

#### Educational Details

| Field           | Instructions                |
| --------------- | --------------------------- |
| Qualifying Exam | 12th/Equivalent             |
| Year of Passing | 2024/2025/2026/Earlier      |
| Board           | CBSE/ISC/State Board        |
| Roll Number     | Your board exam roll number |
| School Name     | Full name of school         |
| School Address  | City, State                 |
| Marks in PCB    | Enter carefully             |

#### Address Details

| Field                  | Instructions         |
| ---------------------- | -------------------- |
| Permanent Address      | Full address         |
| City                   | Town/City name       |
| District               | Select from dropdown |
| State                  | Select from dropdown |
| PIN Code               | 6-digit code         |
| Correspondence Address | Same/Different       |

### Step 3: Upload Documents

#### Photo Upload

1. Click "Upload Photograph"
2. Select file from computer
3. Crop image if needed
4. Preview the upload
5. Confirm if correct

**Common Photo Errors:**
| Error | Solution |
|-------|----------|
| File too large | Compress using online tools |
| Wrong background | Use white/light background |
| Face not visible | Take clearer photo |
| Wrong format | Convert to JPG |

#### Signature Upload

1. Sign on white paper with black pen
2. Scan or photograph clearly
3. Crop to required size
4. Upload in specified format

### Step 4: Choose Exam Centers

#### Selecting Cities

| Selection        | Details                      |
| ---------------- | ---------------------------- |
| Number of Cities | Up to 4 preferences          |
| Priority         | 1st choice is most preferred |
| State Preference | Usually home state first     |
| Availability     | Based on capacity            |

**Tips for Center Selection:**

- Choose cities you can easily travel to
- First preference should be nearest city
- Consider accommodation availability
- Check previous year center locations

### Step 5: Review Application

**Before Proceeding:**

- [ ] Name spelled correctly
- [ ] Date of birth accurate
- [ ] Category selected properly
- [ ] Photo uploaded correctly
- [ ] Signature visible
- [ ] Mobile number correct
- [ ] Email ID correct
- [ ] All educational details accurate
- [ ] Center preferences as desired

### Step 6: Pay Application Fee

1. Click "Proceed to Pay"
2. Select payment mode
3. Complete transaction
4. Wait for confirmation
5. Download receipt

**After Successful Payment:**

- Confirmation page appears
- Status shows "Payment Successful"
- Confirmation SMS/Email received
- Download confirmation page

### Step 7: Print Confirmation Page

1. Click "Print Application Form"
2. Save as PDF for records
3. Take printout
4. Keep safely until admission

### What Can Be Corrected?

| Photo               | Yes         | Pay additional fee     |
| Signature           | Yes         | Pay additional fee     |
| Educational Details | Limited     | Some fields only       |
| Exam Centers        | Yes         | If seats available     |
| Email/Mobile        | Yes         | OTP verification       |

### Correction Fee

| Correction Type      | Fee                |
| -------------------- | ------------------ |
| Photo/Signature      | ₹200               |
| Other Details        | Varies             |

### How to Make Corrections

1. Login with credentials
2. Click "Correction Window"
3. Edit required fields
4. Upload new documents if needed
5. Pay correction fee
6. Submit corrections
7. Download corrected form

## Frequently Asked Questions

### When will NEET 2026 application form be released?

NEET 2026 application is expected to open in February 2026 (second week) on neet.nta.nic.in.

### What is the application fee for NEET 2026?

₹1,700 for General category and ₹1,000 for OBC-NCL, EWS, SC, ST, and PwD categories.

### Can I apply for NEET 2026 without Aadhaar?

For Indian citizens, Aadhaar is mandatory. NRIs and foreign nationals can use passport as ID.

### What if I make a mistake in my application?

You can correct most details during the correction window. Some details like category cannot be changed.

### How many exam centers can I choose?

You can choose up to 4 exam cities in order of preference.

### What happens if payment fails?

Try again with another payment method. If money is deducted, wait 7 days for auto-reversal or contact NTA.

### Can I change my exam center after submission?

Yes, during the correction window if seats are available at the new center.

### What if my photo is rejected?

You'll get a chance to re-upload during correction window with additional fee.
